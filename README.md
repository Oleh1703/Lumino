# Lumino

**Lumino** — інтерактивна веб-платформа для вивчення англійської мови,
розроблена як дипломний проєкт.

## Опис проєкту
Платформа дозволяє користувачам проходити курси, уроки та вправи з англійської мови,
переглядати свій прогрес, працювати зі словником та взаємодіяти з навчальним персонажем Lumi (2D/3D).

Проєкт складається з трьох основних компонентів:
- **Backend API** (ASP.NET Core Web API)
- **Web/Mobile застосунок** для учнів (React)
- **Admin Panel** для керування контентом (React)

## Технології
- Backend: ASP.NET Core Web API (.NET LTS)
- Database: MS SQL Server (Entity Framework Core)
- Frontend: React + Vite
- ORM: Entity Framework Core
- API документація: Swagger
